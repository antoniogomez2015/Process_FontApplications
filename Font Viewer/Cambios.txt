Version 3.86
------------
- Ahora se pueden copiar los archivos asociados a una fuente instalada a una carpeta tambien desde
  la ventana de la lista de fuentes.
- Hechos algunos pequeños ajustes en las operaciones de copiar/mover archivos de fuente.
- Algunos otros pequeños cambios y correcciones.

Version 3.85
------------
- Añadida la opción de de exportar la lista de categorias a una carpeta.
- Añadida la opción de copiar los archivos asociados a una fuente instalada a una carpeta.
- Las categorias son ahora guardadas en el perfil de cada usuario de Windows, de manera de cada
  usuario puede tener su propia lista de categorias. Esto soluciona tambien el problema que
  impedia a usuarios limitados guardar las categorias.
- Algunos otros pequeños cambios y correcciones.

Version 3.82
------------
- Corregido: Al desinstalar una fuente, si estaba marcada la opción de hacer una copia, el programa
  preguntaba por la carpeta de destino en vez de usar la que esta marcada en las opciones.
- Corregido: Algunas de las ventanas del programa no eran reescaladas correctamente cuando Windows
  esta configurado para usar DPI altos (fuentes grandes).
- Algunos otros pequeños cambios y correcciones.

Version 3.80
------------
- La lista con fuentes no instaladas ahora puede ser impresa desde la ventana "Ver carpeta".
- Opcion de hacer una copia de los archivos de fuente antes de desinstalar (borrar) una fuente.
- Añadidas opciones para copiar y renombrar los archivos de fuentes no instaladas.
- Ahora se pueden borrar/mover/copiar todas las fuentes seleccionadas de una vez desde la ventana
  "Ver lista de fuentes".
- Ahora se pueden instalar fuentes temporalmente desde la ventana "Ver carpeta"
- Añadidos varios menús contextuales.
- Mejor deteccion y tratamiento de fuentes no soportadas (por ejemplo, fuentes de bitmap)
- La ventana de impresion ahora permente seleccionar entre centímetros y puldas para los margenes.
- La previsualizacion de carpetas con fuentes ahora puede ser cancelada (para carpetas con muchas
  fuentes puede ser una operación muy larga)
- La barra de estado muestra ahora los caracteres del texto seleccionado en el cuadro de ejemplo
  (util para fuentes de símbolos).
- Algunos otros pequeños cambios y correcciones.

Version 3.70
------------
- Añadido soporte para fuentes Type 1 (*.pfm & *.pfb) (Windows 2000/XP requerido).
- Mejorado el soporte para fuentes TrueType.
- Añdida mas informacion sobre la fuente en las pestañas Fuentes instaladas y

- Corregido: El manejo incorrecto de ciertas fuentes causaba que el programa terminara
  súbitamente.
- Algunos otros pequeños cambios y correcciones.

Versión 3.50
------------
- El código para gestionar fuentes ha sido reescrito, ahora es mas
  escalable y estable.
- Añadida una opcion para importar archivos de categorias (*.LST).
- Mejoradas las opciones de busqueda de fuentes.
- La opción de mostrar caracteres ASCII ha sido mejorada para mostrar

- Algunos otros pequeños cambios y correcciones.

Versión 3.0
-----------
- Interfaz mejorado: Las ventanas de fuentes instaladas y no instaladas
  son ahora una sola ventana con dos solapas. También ahora, la ventana
  principal puede cambiada de tamaño.
- Añadida la opción de borrar fuentes instaladas.
- Añadida la opción de instalar fuentes temporalmente.
- Mas opciones para el manejo de categorías: combinar categorias,
  mostrar fuentes sin categoría, buscar fuentes en categorías.
- Nuevas opciones de manejo de archivos para las fuentes no instaladas:
  Borrar los archivos de fuente o moverlos a una carpeta.
- Arreglados varias fallos en la rutina de instalación de fuentes.
- Arreglados algunos otros pequeños fallos.
